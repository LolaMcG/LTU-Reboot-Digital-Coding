Using basic HTML

What have I learnt?

Today we looked at some different kinds of basic HTML tags and funtions, as well as going over the meaning of tag (the thing that gives a command to the webpage, identifying how you want it to interpret the following data), element (everything that comes inbetween tags), and attribute (this tells us more about an element). Basically, HTML is the 1st layer of a webpage and provides the structure - the structure is very important! Essential for accessability, as well as good for SEO; CSS is the 2nd layer and is for presentation and layout; and the 3rd layer is JacaScript and is for functionality.

Images

If you want to add an image, you must state its source. It's also best if you use an alt attribute, which is the text that will appear to describe a picture if it doesn't load. Even if you leave it blank, it's still best practice to include one.

Links

Links need an anchor tag, the path that the link will take you down, and the text that the user will click.

e.g. Clickhere to open Google. (Clicking this link will override the page you are on and take you to Google in the same tab)

Links in a new tab

To open a link in a new tab you simply add target="_blank" to your anchor tag.

e.g. Click here to open Google in a new tab.

Email links

To add an email link just add mailto: after the href attribute. This will open the default email program on a person's computer. However, apparently this is not the best way to set up emailing as can lead to spam. Don't understand how/why yet!

e.g. Click here to send email.

Phone links

Similar to email links, you just add tele: after the href attribute. You don't need to include dashes or spaces, but you can if it makes it visually easier for you; the browser will still be able to understand the code either way.

e.g. Click here to call now!

Comments

Wrapping text in (but without the space after the !) allows you to leave comments for yourself or other people within your code, without being displayed by the browser. Useful for if you're stepping away from a project for a bit, or working on a group project.

Use in practice?

Followed along in realtime, and have since been doing some basic HTML challenges online which use these skills. As you can see, I've also used them in my CPD here!
